## Запуск

Один общий проект на все лабы (управление через [uv](https://docs.astral.sh/uv/), Python 3.12):

```bash
uv sync                 # из корня — создаст единое окружение для всех лаб
uv run jupyter lab      # либо открыть нужный labN/labN.ipynb в VS Code
```
